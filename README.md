Tech Stack:- Java · JDBC · MySQL · Eclipse IDE

Overview:- 
A Java-based desktop application that automates library operations such as adding, searching, issuing, and returning books.
Built using Core Java, JDBC, and SQL, it ensures seamless data handling and improved system efficiency.

Key Features:-
Manage books and users with full CRUD operations
Transaction management for concurrent operations
Optimized SQL joins and stored procedures (20% faster)
Data validation and error handling for reliability
